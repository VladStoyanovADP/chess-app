This folder should contain SVG images for chess pieces with the following naming convention:

- p.svg - Black pawn
- n.svg - Black knight
- b.svg - Black bishop
- r.svg - Black rook
- q.svg - Black queen
- k.svg - Black king
- P.svg - White pawn
- N.svg - White knight
- B.svg - White bishop
- R.svg - White rook
- Q.svg - White queen
- K.svg - White king

You can use any standard chess piece SVG set.
